📚 منصة زاد المعرفة التعليمية الأزهرية - دليل التشغيل النهائي

الحمد لله تم الانتهاء من تطوير المنصة وربط المساعد الذكي بكفاءة عالية مستخرجاً معلوماته من كتب المنهج المرفقة.

---

## 🔐 أولاً: طريقة استخراج الـ API Key لتشغيل البوت

بسبب القيود الإقليمية المؤقتة لـ Google AI Studio في بعض المناطق، اتبع الخطوات التالية بدقة للحصول على المفتاح مجاناً في ثوانٍ:

1. قم بتشغيل أي برنامج **VPN** على جهازك واضبط الدولة على (الولايات المتحدة أو أي دولة أوروبية).
2. افتح المتصفح وادخل مباشرة على الرابط التالي: [Google AI Studio](https://aistudio.google.com/)
3. سيتيح لك الموقع الدخول فوراً؛ اضغط على زر **"Create API Key"** ثم **"Create API Key in new project"**.
4. انسخ المفتاح (الكود الطويل) واحفظه لديك، وبمجرد نسخه يمكنك إغلاق الـ VPN تماماً، فالمفتاح يعمل على السيرفرات دون الحاجة إليه.

---

## 🛠️ ثانياً: تشغيل المشروع محلياً (على جهازك للتطوير):

1. افتح فولدر المشروع في نظام التشغيل أو الـ Terminal الخاص بـ VS Code.
2. قم بتثبيت الحزم البرمجية المطلوبة عبر الأمر التالي:
   ```bash
   npm install
   قم بإنشاء ملف نصي جديد في جذر المشروع باسم .env وضع بداخله المفتاح الذي قمت بنسخه كالتالي:
   ```

Code snippet
GEMINI*API_KEY=ضع*مفتاحك*الخاص*هنا
لتشغيل السيرفر المحلي والمنصة، اكتب الأمر:

Bash
npm start
افتح المتصفح على رابط: http://localhost:3000

🚀 ثالثاً: الرفع والنشر المباشر على Vercel:
المشروع مجهز كلياً بملفات الإعدادات والمسارات الفورية ولا يحتاج أي تعديل برمجياً.

قم بإنشاء حساب مجاني على موقع Vercel واربطه بحساب GitHub الخاص بك.

اضغط على Add New ثم Project واختر مستودع الكود الخاص بهذه المنصة (zad).

خطوة تفعيل البوت (هام جداً): قبل الضغط على Deploy، انزل لأسفل وافتح قائمة Environment Variables:

في خانة Key اكتب: GEMINI_API_KEY

في خانة Value ضع: (مفتاح الـ API الخاص بك الذي استخرجته).

اضغط على زر Add.

اضغط على زر Deploy الأزرق؛ وخلال أقل من دقيقة سيكون موقعك متاحاً بشكل حي برابط لايف مبروك!
